# اقتراحات تحسين الصفحة الرئيسية — ZU Connect

**تاريخ**: 2026-06-14 | **الحالة**: اقتراحات (لم تنفذ بعد)

---

## تحليل الوضع الحالي

الصفحة الرئيسية حالياً تتكون من 6 أقسام:
1. **Hero section** — صورة حرم جامعي مع عنوان وزرين
2. **Stat cards** — 4 بطاقات إحصاء (2 منها مخفية لغير المسجلين)
3. **Leadership section** — هيكل إداري بخلفية زرقاء وذهبية
4. **News section** — آخر 3 أخبار (النص مخفي لغير المسجلين)
5. **Planner sidebar** — أقرب 4 أنشطة قادمة
6. **AI Chat widget** — مساعد ذكي بسيط بكلمات مفتاحية

### المشاكل الحالية:
- **توزيع غير متوازن**: grid 3 أعمدة يجعل الأخبار في عمودين والنشاطات+الدردشة في عمود واحد — الشاشة غير مستغلة
- **بطاقات الإحصاء**: عند عدم تسجيل الدخول تظهر بطاقتان فقط في grid من 4 أعمدة — شكلة غير منظم
- **لا توجد بطاقات خدمات سريعة** للوصول لأهم الأقسام
- **قسم القيادة**: مكرر (بيانات hardcoded + بيانات API في نفس القسم)
- **AI Chat**: يعتمد على مطابقة كلمات مفتاحية بسيطة فقط
- **الـ Hero**: يمكن أن يكون أكثر تفاعلية وإبهاراً
- **لا يوجد transition سلس** بين الأقسام
- **لا توجد أيقونات تعليمية** في الأقسام المختلفة

---

## الاقتراحات — مرتبة حسب الأولوية

---

### 1. إعادة هيكلة الـ Hero Section  🔥

**الوضع الحالي**: صورة خلفية مع parallax + عنوان + زرين.

**التحسين المقترح**:
- إضافة طبقة **gradient animated** متحركة بدلاً من الثابتة
- إضافة **Lottie animation** تعليمية (كتاب/تخرج/شهادة) إلى جانب النص
- جعل النص **متحركاً** بحرفية عند التحميل (typewriter effect للحروف)
- إضافة **إحصائيات عائمة** (عدد الطلاب المسجلين حالياً، عدد المتصلين)
- إبراز زر **تسجيل الدخول** للمستخدمين غير المسجلين
- إضافة الـ **Logo** بشكل بارز بدلاً من الاكتفاء به في الـ Topbar

**التنفيذ**: استخدام `framer-motion` أنيميشن للمكتوب + LottieAnimation التوضيحية الموجودة.

---

### 2. إضافة Quick Service Cards 🚀

**الوضع الحالي**: لا يوجد وصول سريع للخدمات الرئيسية.

**التحسين المقترح**: إضافة قسم **بطاقات خدمات تفاعلية** بعد الـ Hero مباشرةً:
- **المكتبة الرقمية** 📚
- **الدورات التدريبية** 🎓
- **الأخبار** 📰
- **التواصل** 💬
- **الأنشطة** 📅
- **المقترحات** 💡

كل بطاقة تحتوي على: أيقونة + عنوان + وصف مختصر + رابط للصفحة.
تظهر 3-6 بطاقات في Bento Grid شبيه بصفحة الخدمات.

**التنفيذ**: استخدام `card.tsx` الموجود مع `framer-motion` hover effects.

---

### 3. تحسين بطاقات الإحصاء 📊

**الوضع الحالي**: بطاقتان لغير المسجلين في grid 4 أعمدة.

**التحسين المقترح**:
- جعل الـ grid **ديناميكياً**: `grid-cols-2 md:grid-cols-[repeat(auto-fit,minmax(150px,1fr))]`
- للزوار: عرض 3 بطاقات أساسية (طلاب، كليات، خدمات)
- للمسجلين: عرض 6 بطاقات (طلاب، كليات، أنشطة، ملفات، دورات، أعضاء)
- إضافة **أيقونات متحركة** عند التمرير
- إضافة **شريط تقدم دائري** (Circular Progress) بدلاً من الأرقام فقط — أو كمكمل

**التنفيذ**: `useCountUp` موجود — فقط تعديل الـ grid ليكون متجاوباً.

---

### 4. Bento Grid للأقسام الرئيسية 🧩

**الوضع الحالي**: grid 3 أعمدة غير متوازن.

**التحسين المقترح**: استخدام Bento Grid مثل:
```
┌─────────────────┬──────────────┐
│    الأخبار      │  الأنشطة     │
│    (2 أعمدة)    │  القادمة     │
│                 │  (1 عمود)    │
├────────┬────────┴──────────────┤
│ الكورس │  AI Chat  │  الإحصاءات│
│ المميز  │ (مصغر)    │  السريعة  │
└────────┴───────────┴───────────┘
```

هذا يسمح بعرض أكثر في نفس المساحة مع تنظيم أفضل.

**التنفيذ**: استخدام CSS Grid مع `col-span-*` classes.

---

### 5. تحسين AI Assistant 🤖

**الوضع الحالي**: محادثة محلية تعتمد على if/else.

**التحسين المقترح**:
- ربط الـ AI Chat مع **API خارجي** (OpenRouter — مذكور في `.local/skills/ai-integrations-openrouter`)
- إضافة **أسئلة مقترحة** (Suggested questions) كأزرار صغيرة
- إضافة **حالة الكتابة** (Typing indicator)
- إضافة **سياق المستخدم**: إذا كان مسجلاً، استخدم اسمه في الترحيب
- تحسين تصميم الـ chat ليكون مثل ChatGPT UI (مع markdown support)

**التنفيذ**: استخدام icons الموجودة لأسئلة مقترحة، Lottie لـ typing indicator.

---

### 6. تحسين قسم القيادة 👥

**الوضع الحالي**: بيانات Hardcoded + API منفصل.

**التحسين المقترح**:
- جلب كل البيانات من API (`/api/leadership`) — لا hardcoded data
- إضافة **صور افتراضية** مع الحروف الأولى (initial avatar) لكل عضو
- إضافة **ترتيب هرمي** best visualized
- تصميم **كرت رئيسي** للرئيس + كروت فرعية للنواب + grid للأعضاء
- إضافة hover effects مع **popover** يظهر معلومات أكثر

**التنفيذ**: استخدام `avatar.tsx` الموجود مع تصميم شبيه للقيادة.

---

### 7. إضافة Social Proof & Quick Stats 📈

**الوضع الحالي**: لا يوجد دليل اجتماعي.

**التحسين المقترح**:
- إضافة **شريط إنجازات** (عدد المستخدمين المسجلين، عدد الدورات المكتملة، إلخ)
- إضافة **testimonials** أو اقتباسات من طلاب حقيقيين (Carousel)
- إضافة **عدد المتصلين حالياً** أو النشاطات الحية

**التنفيذ**: استخدام `carousel.tsx` الموجود.

---

### 8. تحسينات الـ UX العامة 🎯

| المشكلة | الحل |
|---------|------|
| لا يوجد تمييز واضح بين الأقسام | إضافة **Section dividers** مع أيقونات وألوان متدرجة |
| Stat cards layout مكسور عند عدم تسجيل الدخول | استخدام `auto-fill` و `minmax` في CSS Grid |
| الـ Hero طويل جداً (60vh) وتستهلك مساحة | تصغير إلى `40vh` أو إضافة عناصر تثبيت |
| لا يوجد scroll indicator | إضافة **سهم متحرك** أسفل الـ Hero |
| أخبار home تظهر رابط `news/id` رغم أن الـ route غير موجود | إزالة `<Link>` أو إضافة route |
| اللون الأزرق الداكن (#0b1f3f) غير متناسق مع الثيم | استخدام `bg-card` مع التدرجات بدلاً من الألوان الثابتة |
